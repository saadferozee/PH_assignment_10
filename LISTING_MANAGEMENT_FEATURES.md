# Listing Management Dashboard

## Overview
Comprehensive listing management system for AdoptyCo admin dashboard with vendor analytics, order tracking, and detailed product information management.

- `ManageListings.jsx` - Main listing management page

- Calculates vendor statistics in real-time
- Cross-references orders with listings for analytics

## Usage Instructions

### For Administrators
1. **Access**: Navigate to `/dashboard/manage-listings`
2. **View Stats**: Check top stats cards for overview
3. **Search**: Use search bar to find specific listings
4. **Filter**: Use category and price filters to narrow results
5. **View Details**: Click eye icon for comprehensive listing information
6. **Manage**: Delete listings with confirmation dialogs

### Vendor Analytics
- **Vendor Performance**: See total listings, orders, and revenue per vendor
- **Product Performance**: Track individual product success
- **Market Insights**: Identify popular categories and pricing trends

## Key Metrics Tracked

### Platform Level
- Total active listings
- Number of unique vendors
- Total orders processed
- Revenue generated
- Category distribution

### Vendor Level
- Listings per vendor
- Orders per vendor
- Revenue per vendor
- Success rate analysis

### Product Level
- Orders per product
- Revenue per product
- Performance indicators
- Popularity metrics

## Security Considerations
- Admin-only access through PrivateRoute
- Confirmation dialogs prevent accidental deletions
- Proper error handling for failed operations
- Safe data processing with null checks
- Vendor privacy protection
